This sample is a modified MeshCreation sample for GameBase / Emergent's Gamebryo Lightspeed 3.2, which replaces the ocean included in the demo with Triton. It supports both DirectX9 and DirectX11.

In order for the solution file to load, these files need to be copied into:

C:\Emergent\Gamebryo-LightSpeed-3.2-Binary\Samples\GraphicsTechDemos\MeshCreation

(substituting your own installation path for Lightspeed.)

Be sure to back up the original MeshCreation sample first.

Once in place, you'll find a solution file in the Win32/VC90 directory.

If you run into any trouble, contact support@sundog-soft.com and we'll be happy to help.
