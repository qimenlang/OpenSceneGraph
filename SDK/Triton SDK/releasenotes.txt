Triton Release History

Version 4.65 March 13, 2026
- Fix GLSL error in god rays shader

Version 4.64 December 7, 2025
- Added support for Visual Studio 2026 (new vc145 libraries & DLL's)
- Updated to CUDA 13.1 (you may need to update your NVidia drivers!)

Version 4.63 June 26, 2025
- Further fixes for jagged artifacts when using use-mesh=no and 
  ellipsoid-double-precision=yes in WGS84 environments. This is a common
  configuration when using extremely high zoom levels.

Version 4.62 June 25, 2025
- Fix for ellipsoid-smooth-horizon setting
- Added adjust-mesh-position-for-zoom config setting (off by default.) This
  recenters the radial grid to center it along where the near clip plane
  intersects the ocean, focusing its density where it is needed the most.
  This can help a little when dealing with high zoom levels. Be sure to
  use this in conjunction with ellipsoid-double-precision=yes.
- Fix unnecessary precision loss in geocentric geometry when using
  ellipsoid-double-precision=yes. This avoids jaggy artifacts when running
  with use-mesh=no in WGS84 mode.

Version 4.61 June 11, 2025
- Eliminate use of gl_FragColor outside of user-functions.glsl, enabling use of
  gl_FragData[] for MRT when using OpenGL 2.0 mode

Version 4.60 May 1, 2025
- ARM64 support for Windows 11 (offered as a separate SDK download) - limited 
  to OpenGL / OpenCL only
- Fix shader errors under Microsoft OpenGL compiler


Version 4.59 January 21, 2025
- Disable UBOs on AMD cards when using OpenGL versions below 4.5.

Version 4.58 October 31, 2024
- Added user_decal_depth hook into user-functions.glsl to allow modifying the
  depth values written by decals.
- Added config setting bypass-override-position-on-depth-pass (on by default)
  which prevents custom code for log depth buffers etc. from affecting the
  depth texture used by decal effects
- Note that if you are integrating a log depth buffer or similar scheme, you
  will need to specify your depth function in both overridePosition in
  user-vert-functions.glsl and user_decal_depth in user-functions.glsl.
  (only supported in OpenGL)

Version 4.57 September 27, 2024
- Fix spray effects when using patch shaders

Version 4.56 September 13, 2024

  no simulated wind

  wave model

Version 4.55 September 12, 2024

  older CUDA releases (for best driver compatiblity)
- Fix crash in Ocean::ClearFFTInputArray()
- Fix memory leak if Ocean::OverrideFFTInputArray() is called repeatedly

Version 4.54 December 5, 2023
- Ship wakes now affect height queries from Ocean::GetHeight! This is based on a parallel
  simulation of ship wakes on the CPU based on summed circular waves, which may not 
  exactly match the visual representation, but is still realistic. If you don't need
  this feature, it may be disabled with wakes-affect-height-queries in Triton.config.
  You can also affect its fidelity with the wake-wave-generation-period setting.
- Fix distance height queries are affected by circular waves
